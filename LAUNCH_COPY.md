### Body
```
We built an MCP suite that gives Claude real business intelligence frameworks rather than just API access.

Two packages live on npm today:

@autostackup/sales — lead qualification (BANT + MEDDIC dual scoring), ICP fit scoring with disqualification, weighted pipeline forecasting with stale/at-risk detection, and multi-channel outreach generation

The key design decision: these are strategic reasoning tools, not API wrappers. Most tools work with zero external API keys — it's Claude applying the framework to your inputs.

Source: github.com/autostackup
npm: npmjs.com/~autostackup

@autostackup/hr (recruitment screening, onboarding protocol) and @autostackup/founder (full orchestrator from concept to launch) are next in the roadmap.
```

```
- `qualify_lead` — BANT + MEDDIC dual scoring → gives you tier (A/B/C/D), gaps, and exact next step
- `score_icp` — Score any prospect against your Ideal Customer Profile, auto-disqualifies on must-haves
- `track_pipeline` — Weighted forecast, stale deal detection, at-risk flags, urgent action list
- `generate_outreach` — Email/LinkedIn/SMS copy with 4 subject variants + follow-up pre-written
- `enrich_lead` — Hunter.io or Apollo.io contact enrichment (BYOK)

**@autostackup/marketing** — 3 tools:
- `kotler_analysis` — Scores your company across all 9 Kotler elements, priorities the gaps
- `plan_imc_campaign` — Full campaign plan across Meta + Google with budget splits, audiences, creative brief, launch checklist
- `segment_audience` — Persona map, messaging matrix, channel recs

Setup is 2 minutes — add to Claude Desktop config and restart.

Most tools need zero API keys. Open source, MIT.

**npm:** npmjs.com/package/@autostackup/sales

Would genuinely love to know what framework you'd want Claude to run for you next — building HR and the founder orchestrator now.
```
